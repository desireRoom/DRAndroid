引体向上 共同注意事项：

  • 起始姿势：肘部微微弯曲，有两个好处，第一，避免肘部过度伸展，减少肘部承受的压力。第二可以帮助你稳住上身来左强力的动作。
  • 在动作最低点，身体绝对不要完全放松、彻底伸展，因为这样会把压力从肌肉转到连接关节的韧带上。在最低点不仅肘部要弯曲大约 10度左右，为了稳住上身，肩部也要保持“收紧”状态
  • 理想的动作幅度：手臂几乎伸直(微微弯曲)，把身体上拉至下巴超过横杆。不要继续上拉，不然会限制力量发挥，也使上肢处于容易受伤的位置。
  • 收紧双肩在任何悬垂练习中都对保护肩部有重要意义
  • 抓握姿势无所谓，选一个自己最舒服的来，可以尝试使用吊环，帮助手臂找到最自然最佳的抓握姿势。
  • 蹬腿会跟关节带来不必要的压力，造成一种很有力量的感觉，动作也容易走形。初学者千万不要蹬腿，保持2-1-2规则，
  • 在做第五式以上的水平时，可以先做严格规范的动作，尽量多做，至少三四次，到几乎力竭时可以借助蹬腿产生的力量多做一两次。但是不要用过头。
  • 当你想升级到下一式时，只有不蹬腿所做的次数才算数。
  • 慢慢来，不要急于从一个动作过渡到下一个动作，这一条适用于所有的自重练习。切记，升级到难度更高的动作，只是表明你的力量增加了而已。而这些增加的力量，只是通过日积月累地重复难度较低地动作才实际获得地，加急升级并不能使你更快的获得力量。不要自己骗自己。
